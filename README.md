An autonomous fire-fighting robot using IR flame sensors, DC motors, a water pump, and a servo-based nozzle.

### Features
- Flame detection using 3 IR sensors (Left, Front, Right)
- Directional movement control using L298N motor driver
- Servo-controlled water spray mechanism
- Relay-controlled water pump

### Tech Used
- Arduino Uno
- C / Embedded C
- IR Flame Sensors
- DC Motors + Motor Driver
- Servo Motor & Relay
- DC Pump
